# Customer Churn Prediction

This project predicts customer churn using machine learning.

## Technologies
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn

## Project Structure
data/ - dataset  
notebooks/ - analysis notebook  
src/ - python scripts

## Run the Web App
Install dependencies:
pip install -r requirements.txt
Run the app:
streamlit run app.py
